## Descripción

Pantalla de creación/edición de una plantilla de consentimiento informado. Incluye breadcrumb de navegación, formulario con título y descripción, selector de procedimientos vinculados (badges), y un editor de texto enriquecido para el cuerpo del consentimiento.

## Secciones principales

- **Breadcrumb:** 5× `Breadcrumb / BreadcrumbItem` + `Icon / ChevronRight` / `Breadcrumb Chevron` — navegación jerárquica.
- **Formulario:** `Title` (input de título), `Section Description`, `Input search` para vincular procedimientos.
- **Procedimientos vinculados:** 5× `Badge` (uno por procedimiento), cada uno con `Procedure label`.
- **Editor de texto ("Consent text"):** toolbar de formato básico (B, I, U, H1, H2, ¶, lista con viñetas, lista numerada) — nota en UI: "Basic formatting only".
- **Alertas:** 3× `Alert Icon` (contextuales, ej. advertencias sobre el contenido legal).
- **Acciones:** 3× `Button` (probablemente Cancel / Save Draft / Publish o similar).
- **Contacto:** `Contact Icon` + `Input`.
