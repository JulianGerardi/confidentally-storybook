# Radiography — flujo completo

- **Archivo Figma:** UX-UI · 2.0
- **Página:** ↪ Radiography (node `781:5413`)
- **URL base:** https://www.figma.com/design/aESBq2SDwG2b4fnTyfBUgn/UX-UI-·-2.0?node-id=781-5413
- **Tamaño de cada paso:** 1064×910.6

## Descripción

Flujo de 4 pasos para subir y revisar radiografías de un paciente, todos bajo el frame "Radiography - List" (nombre reutilizado para cada paso del wizard).

## Paso 1 — Listado (node `824:1541`)

- Header + `ClinicalModeButton` (tabs de navegación clínica)
- Grilla de miniaturas con `Calendar Icon` (32 instancias — imágenes con fecha)
- 2× `Button` de acción

## Paso 2 — Subida de archivo (node `824:1729`)

- Zona de "Drag and Drop" (Título + Subtítulo)
- Campo `Label` + `Placeholder` (varios inputs de metadata: procedimiento, etc.)
- `Icon / FileText` ×5 (archivos adjuntos), botón `btn` de confirmar/cancelar por archivo

## Paso 3 — Visor con hallazgos (node `824:2060`)

- Visor de imagen con `Zoom In` / `Zoom Out` / `Expand Icon`
- Panel lateral de "Findings" (`Findings Label`, lista de `Badge` con fecha)
- `Save Button`

## Paso 4 — Detalle completo (node `844:5408`)

- Igual al Paso 3 más `Sidebar / Header Button`, `Steps` (stepper de progreso) y `Container` ×6 (tarjetas de hallazgo)

## Notas

- El frame "Radiography - List (Label)" (`824:2944`) es solo un separador de sección dentro del canvas, no una pantalla.
- Reutiliza `ClinicalModeButton`, `Badge`, `Calendar Icon` — ya documentados en las páginas de componentes.
