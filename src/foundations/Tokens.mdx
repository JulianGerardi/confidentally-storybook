# Design tokens

Tokens extraídos de **Design system · 2.0** (Figma) — fuente: `src/tokens/tokens.json` / `src/tokens/tokens.css`.
Se regeneran corriendo `FIGMA_TOKEN=... npm run sync-tokens` (ver `scripts/sync-figma-tokens.mjs`).

## Color

<div style={{ display: "grid", gridTemplateColumns: "repeat(auto-fill, minmax(160px, 1fr))", gap: 12 }}>
  {Object.entries(tokens.color).flatMap(([group, value]) =>
    typeof value === "string"
      ? [[group, value]]
      : Object.entries(value).filter(([k]) => k !== "figmaName").map(([k, v]) => [`${group}.${k}`, v])
  ).map(([name, hex]) => (
    <div key={name} style={{ border: "1px solid #e4e4e7", borderRadius: 8, overflow: "hidden" }}>
      <div style={{ background: String(hex), height: 56 }} />
      <div style={{ padding: "6px 8px", fontFamily: "monospace", fontSize: 11 }}>
        <div>{name}</div>
        <div style={{ color: "#71717a" }}>{String(hex)}</div>
      </div>
    </div>
  ))}
</div>

## Radius

<div style={{ display: "flex", gap: 16, flexWrap: "wrap", alignItems: "flex-end" }}>
  {Object.entries(tokens.radius).filter(([k]) => k !== "figmaName").map(([name, value]) => (
    <div key={name} style={{ textAlign: "center" }}>
      <div style={{ width: 64, height: 64, background: "#0056ef", borderRadius: String(value) }} />
      <div style={{ fontFamily: "monospace", fontSize: 11, marginTop: 4 }}>{name}: {String(value)}</div>
    </div>
  ))}
</div>

## Tipografía

- Heading: **Manrope** (`tokens.typography.fontFamilies.heading`)
- Body: **Inter** (`tokens.typography.fontFamilies.body`)
- Base/small: {tokens.typography.baseSizes.small.fontSize} / {tokens.typography.baseSizes.small.lineHeight} line-height
